VerityNgn — Open Source

The first open-source multimodal pipeline for claim-by-claim verification of YouTube videos. Counter-intelligence pass hunts contradictory primary evidence. Runs locally with your own API keys.

pip install verityngn verityngn analyze "https://www.youtube.com/watch?v=VIDEO_ID" # Local file (depositions, court video) verityngn analyze --file clip.mp4 --title "Expert excerpt"
  • Full multimodal analysis (video, audio, OCR)
  • Transparent research methodology
  • 100% open source (Apache 2.0)
  • URL safety filter for citation links
